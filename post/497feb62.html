MySQL常用操作

  1. 常用操作

    1. MySQL服务器
      1. 启动
      2. 停止
      3. 登录
    2. 数据库
      1. 新建
      2. 修改
      3. 删除
    3. 表
      1. 新建
      2. 修改
      3. 删除

  2. 数据库操作方式

    1. 操作MySQL数据库有以下两种方式
      1. 命令方式
        1. 用管理员身份打开cmd控制面板
        2. 输入命令 mysql -u root -p
        3. 输入你设置的密码即可打开
      2. MySQL图形
        1. 使用工具Navicat Premium

  3. 启动/停止MySQL服务器

    1. 命令行方式
      1. net start mysql80
      2. net stop mysql80
    2. 使用图形服务工具
      1. 略

  4. 数据库整体操作

    1. 创建数据库

      1. CREATE DATABASE [IF NOT EXISTS] <数据库名>
      2. [[DEFAULT] CHARACTER SET <字符集名>]
      3. [[DEFAULT] COLLATE <校对规则名>];

    2. 查看或显示数据库

      1. show databases [like ‘数据库名’];

    3. 选择数据库

      1. use 数据库名;

    4. 修改数据库

      1. ```mysql
        alter database [数据库名] {
        [ default ] character set <字符集名> |
        [ default ] collate <校对规则名>}

    5. 修改表结构

      1. alter table <表名> [修改选项];
修改选项的语法格式如下：
{ add column <列名> <类型>
| change column <旧列名> <新列名> <新列类型>
| alter column <列名> { set default <默认值> | drop default }
| modify column <列名> <类型>
| drop column <列名>
| rename to <新表名>
| character set <字符集名>
| collate <校对规则名> }

      2. 删除外链约束

        1. alter table <表名> drop foreign key <外键约束名>;
